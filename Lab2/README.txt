DESCRIPTION

This lab implements 4, four-bit registers that stores values on a kaypad and performs a left arithmetic shift with Arithmetic Logic Unit buit out of gates, Multiplexors, and Data Flip Flops.

INSTRUCTIONS

This program is intended to be run on Multimedia Logic (MML). Select clear to initialize registers to 0. Select store to choose which value to store then selct adress of which Register to store value to. ALU performs a bitwise left arithmetic shift of ALU Input 2 by the amount in ALU input 1
